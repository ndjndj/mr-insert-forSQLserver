## 使い方

1. テーブル名を入力する
    - table name

2. tsv を貼り付ける
    - paste area

3. html table に変換する
    - CONVERT TABLE 

4. 文字列にしたい列を選択する
    - ''で囲む

5. SQL に変換
    - CREATE INSERT STATEMENT
